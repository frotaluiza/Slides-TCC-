\begin{slide}{Análise Exploratória}

        \item Pearson: T$_{out}$ têm correlações lineares fortes ($|r|>0,9$); fluxo moderado ($|r|\approx0,7$)
        \item Complexidade do fluxo $\rightarrow$ justifica modelagem multivariada
      \end{Item}

  \note{A análise exploratória mostra que as temperaturas de saída têm forte correlação linear com variáveis térmicas, enquanto o fluxo de permeado apresenta comportamento mais complexo, dependendo de múltiplas variáveis. Isso motiva a modelagem multivariada. [~1 min]}
\end{slide}

\begin{slide}{O que a Hibridização está Corrigindo?}
  \begin{columns}
    \begin{column}{.55\linewidth}
        \begin{Item}
          \item OLS explica $>95\%$ da variância de todos os targets $\rightarrow$ região aproximadamente linear
          \item Modelos não lineares puros (árvores) têm desempenho \textbf{pior} que OLS
          \item \textbf{Conclusão:} o ganho híbrido não vem de capturar não linearidades, mas de \textbf{corrigir o viés} dos modelos puramente orientados por dados
          \item A informação física (modelo 0D) fornece uma referência fenomenológica que reduz o erro sistemático da MLP pura, especialmente quando os dados são escassos (174 amostras)
        \end{Item}
    \end{column}
    \begin{column}{.45\linewidth}
      \includegraphics[width=\linewidth]{tcc_images/chap06/overlay_ZohanResidual_Flux.png}\\
      \centering\small Linha y=x: concordância ideal (predito = experimental)
    \end{column}
  \end{columns}
  \note{O ganho dos híbridos não é capturar relações não lineares — a região é aproximadamente linear (OLS com R² > 0,95). O ganho é corrigir o viés dos modelos data-driven usando a física como referência. A linha y=x representa predito = experimental. [~1 min]}
\end{slide}
