\begin{slide}{Etapa 2.1: Preparação dos Dados — Técnicas de Escalonamento}
  \small
  \begin{Item}[0.1]
    \item \textbf{Padronização Z-score}: $z = (x - \mu) / \sigma$ $\rightarrow$ média 0, desvio 1 — \textit{adotada}
    \item \textbf{Normalização Min-Max}: $x' = (x - x_{min}) / (x_{max} - x_{min})$ $\rightarrow$ escala $[0,1]$
  \end{Item}
  \vspace{0.15cm}
  \begin{Item}[0.08]
    \item \textbf{Z-score} ($\mu=0$, $\sigma=1$): cada variável passa a ser medida em "desvios padrão" — preserva a forma da distribuição e funciona bem mesmo com valores extremos
    \item \textbf{Min-Max} ($[0,1]$): comprime os dados num intervalo fixo — \textbf{se um outlier distorce o mínimo ou máximo, os demais pontos são esmagados} numa fração do intervalo, perdendo poder de distinção. Além disso, não centraliza em zero, o que penaliza coeficientes em Ridge/Lasso
  \end{Item}
  \vspace{0.05cm}
  {\small\textit{Implementação:} escalonamento dentro do \texttt{Pipeline} — ajustado só no treino de cada fold, sem vazar info para a validação}
  \note{Z-score vs Min-Max: o Z-score mede cada variável em desvios padrão, preservando a forma dos dados. Min-Max comprime tudo para [0,1] — um outlier distorce a escala e os dados normais perdem poder de separação. Além disso, Ridge/Lasso precisam de centragem em zero. [~1,5 min]}
\end{slide}

\begin{slide}{Etapa 2.3: Preparação dos Dados — Seleção de \textit{Features} e Discretização}
  \small
  \begin{columns}[t]
    \begin{column}[t]{.5\linewidth}
      \textbf{Seleção de atributos}
      \begin{Item}[0.15]
        \item \textbf{Justificativa:} verificar se o conjunto de 5 preditores selecionados por critério físico poderia ser simplificado sem perda de desempenho
        \item \textbf{Método:} busca exaustiva (\textit{Best Subset Selection}) avaliando todas as $2^5 = 32$ combinações de variáveis com regressão linear multialvo e GroupKFold
        \item \textbf{Resultado:} as configurações com melhor desempenho incluíam todos os 5 preditores — a seleção física inicial foi consistente com a análise baseada em dados
      \end{Item}
    \end{column}
    \begin{column}[t]{.5\linewidth}
      \textbf{Discretização (\textit{Binning})}
      \begin{Item}[0.15]
        \item \textbf{Justificativa:} os experimentos seguem um planejamento fatorial, produzindo regiões de dispersão bem delimitadas para cada variável de entrada — a discretização poderia capturar essas faixas operacionais
        \item \textbf{Método:} mistura Gaussiana unidimensional (critérios AIC/BIC) para definir os intervalos; $2^5 = 32$ configurações contínua vs discretizada avaliadas com regressão linear multialvo
        \item \textbf{Resultado:} nenhuma configuração discretizada apresentou melhoria consistente para os 3 alvos simultaneamente — optou-se por manter a forma contínua original
      \end{Item}
    \end{column}
  \end{columns}
  \note{Tanto a seleção de atributos quanto a discretização foram avaliadas por busca exaustiva com regressão linear multialvo. Nenhuma das estratégias ofereceu ganho consistente para os 3 alvos, confirmando que a seleção física inicial dos preditores e a forma contínua das variáveis eram adequadas. [~1,5 min]}
\end{slide}

\begin{slide}{Etapa 3: Critérios de Seleção e Métricas de Erro}
  \small
  \begin{columns}
    \begin{column}{.5\linewidth}
      \textbf{Critério hierárquico:}
      \begin{Item}[0.15]
        \item \textbf{Nível 1 — Dentro de cada família:} 1-SE + mínima complexidade para escolher hiperparâmetros
        \item \textbf{Nível 2 — Entre famílias:} comparação direta pelo menor RMSE$_{CV}$ no fluxo
      \end{Item}
      \textbf{Métricas de erro}
      \begin{Item}[0.15]
        \item \textbf{Erro de generalização:} média do RMSE entre os 3 folds do GroupKFold — guia a busca de hiperparâmetros e seleção dentro de cada família
        \item \textbf{Erro de teste (hold-out):} RMSE calculado no hold-out de 35 amostras — métrica final para comparar modelos selecionados
      \end{Item}
    \end{column}
    \begin{column}{.5\linewidth}
      \textbf{Seleção por target:}
      \begin{Item}[0.15]
        \item Cada variável de saída (Flux, T$_{alim,out}$, T$_{ref,out}$) pode ter um modelo diferente
        \item A busca de hiperparâmetros é otimizada separadamente para cada alvo
        \item Ao final, cada target tem seu próprio modelo selecionado pelo critério 1-SE
      \end{Item}
      \vspace{0.3cm}
      \textbf{Hold-out:}
      \begin{Item}[0.15]
        \item 20\% dos dados (35 amostras) separado \textbf{antes} de qualquer modelagem
        \item Contém amostras dos 3 regimes de salinidade
        \item O RMSE no hold-out estima o desempenho em condições operacionais representativas
      \end{Item}
    \end{column}
  \end{columns}
  \note{O critério de seleção tem dois níveis. As métricas de erro são: o erro de generalização (média do RMSE dos 3 folds) guia a busca de hiperparâmetros; o erro de teste (hold-out) é a métrica final. A seleção por target permite que cada variável de saída tenha o modelo mais adequado. [~1,5 min]}
\end{slide}
